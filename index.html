Nuestro objetivo

Lograr crear un servidor autosustentable capaz de proveer servicios web en areas remotas.

Hoy en día, la dependencia del internet o de servicios web es necesaria para facilitar y hacer eficientes distintas tareas que llevamos a cabo todos los días.

Por tal motivo, desarrollaremos el servidor autosustentable, con el fin de que este sea capaz de distribuir servicios web básicos y que con ayuda de paneles solares, pueda recolectar la energía para su funcionamiento, intentando optimizar la energía recibida.

Tecnologías comunes para la problemática

Existen diversas tecnologías para proveer acceso a internet en áreas remotas, proyectos tales como watly, Google Loon y WIIMAX.
Dado que el costo de estas tecnologías es un poco elevado para un propósito específico, nuestra idea es dar una solucion economica y fiable a esta problemática.

Diseño

Para la realización de este proyecto, hemos decidido usar un arduino Yun, el cual, dentro de sus propiedades, viene con 2 procesadores. Adicionalmente, el Yun, viene con una distribución de Linux instalada y con una interfaz wireless, la cual, usaremos para hacer la distribución de servicios web, mientras que el microcontrolador del arduino, se encargará de optimizar la energía recibida por los paneles solares. Esto lo lograremos moviendo los paneles solares con servo motores.

Otros principios utilizados

Adicionalmente, en el proyecto hemos tenido en cuenta otros componentes como el servomotor y el uso del acumulador que el sistema necesitaría para funcionar en ausencia de el sol.

Principalmente, el servomotor tiene una propiedad indispensable para el funcionamiento de este proyecto gracias a que trabaja con grados, esto lo hace midiendo el ángulo de su eje con un potenciómetro que alimenta un controlador que indica que tanto debe girar el motor DC integrado para alcanzar los ángulos deseados.

Respecto al acumulador, funciona por 2 procesos quimicos conocidos como oxidacion y reduccion, durante su descarga, el material activo del electrodo positivo se oxida, liberando electrones, y el material del electrodo negativo es reducido, captando dichos electrones. Estos electrones constituyen el flujo de corriente eléctrico que atraviesa el circuito externo.
